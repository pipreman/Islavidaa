Installation

To install the Marktify script, your system must have set up of Laravel. If you are new to this then please check the section on How to Set Up Laravel on your system.

How to Set up Laravel

How To Set Up Laravel on your server.

Before starting please make sure that your system fulfill the following server requirements for set up

1) Required PHP Version : PHP >= 8.1

2) PHP configuration : Open your php configuration file php.ini and change the following settings.

  • max_execution_time = 30000
  • max_input_time = 30000
  • memory_limit = 512M
  • post_max_size=100M
  • Upload_max_filesize = 128M

3) Make sure to all extensions are available.

  • extension=curl
  • extension=fileinfo
  • extension=gd
  • extension=gettext
  • extension=intl
  • extension=mbstring
  • extension=exif
  • extension=mysqli
  • extension=pdo_mysql
  • extension=pdo_sqlite
  • extension=zip

4)Folder Permission : Please check folder permission meets the mentioned requirements.

  • .env : 644
  • bootstrap/ : 755
  • bootstrap/cache/ : 755
  • storage/ : 755
  • storage/logs/ : 755
  • storage/framework/: 755

Step 1 - Download executable file of composer. You can download it from the home page of composer.

Step 2 - Launch the exe file you downloaded.

Step 3 - Follow the setup wizard and proceed with the installation.

Step 4 - After the successfull installation open the command prompt and enter a command composer you will get this result.

How to Update Script

Note : It is recommended to take a backup before updating the script. If you have made any changes in the code, then on updating the script those changes gets removed. So please keep the backup of those files/folders.

How to update script to Version v4.0

->Override folder app, config, database, public/admin-theme, public/user-theme , resources ,routes, vendor, composer.json, composer.lock from the latest version.

->Add the following parameter in the env file

# AWS S3 Configuration

  • AWS_ACCESS_KEY_ID=
  • AWS_SECRET_ACCESS_KEY=
  • AWS_DEFAULT_REGION=
  • AWS_BUCKET=
  • AWS_URL=
  • AWS_ENDPOINT=
  • AWS_USE_PATH_STYLE_ENDPOINT=

# Wasabi Configuration

  • WASABI_KEY=
  • WASABI_SECRET=
  • WASABI_REGION=
  • WASABI_BUCKET=
  • WASABI_ENDPOINT=

# Flutterwave Key

  • FLUTTERWAVE_KEY=
  • FLUTTERWAVE_SECRET=

# Stripe key

  • STRIPE_KEY=
  • STRIPE_SECRET_KEY=

# Pawapay token

  • PAWAPAY_TOKEN=

-> After updating the script run the following command

  • composer require league/flysystem-aws-s3-v3

How to update script to Version v3.0

->Override folder app, config, database, public/admin-theme, public/user-theme , resources ,routes, vendor, composer.json, composer.lock from the latest version.

->Add the following parameter in the env file

  • RAZORPAY_API_KEY=
  • RAZORPAY_API_SECRET=
  • GOOGLE_CLIENT_ID=
  • GOOGLE_CLIENT_SECRET=
  • FACEBOOK_CLIENT_ID=
  • FACEBOOK_CLIENT_SECRET=

-> After updating the script run the following commands

  • php artisan optimize:clear
  • php artisan migrate
  • php artisan db:seed AuthorPageSeeder

How to update script to Version v2.0

->Override folder app, config, database, public/admin-theme, public/user-theme , resources ,routes, vendor, composer.json, composer.lock from the latest version.

->Add the following parameter in the env file

  • RAZORPAY_API_KEY=

  • GOOGLE_CLIENT_ID=
  • GOOGLE_CLIENT_SECRET=
  • FACEBOOK_CLIENT_ID=
  • FACEBOOK_CLIENT_SECRET=

-> Replace the old tp_migration table with new in the database.

-> After updating the script run the following commands

  • php artisan optimize:clear
  • php artisan migrate
  • php artisan db:seed --class=EmailProvidersSeeder

Admin Dashboard

Admin Dashboard comes with the various backend settings to flexibly appearance and operation of the website as well as can manage Users, Products, Vendors and many more.


My Board

My Board settings contains the counts of Total Users, Active Users, Deactive Users, Active Products, Free Products, Total Registrations, Total Product Views and many more information. With this you can check the Product Views On Devices, Product Views From Browser,User Registration, and Order Sale in graphical format.



Products

Admin can manage products through this settings. It consist of the three sub settings : Categories, Sub Categories, and Products.

Categories

With this setting admin can add new categories or can manage the existing. In manage catagories admin can edit or delete the categories.



Sub Categories

With this setting admin can add new sub categories or can manage the existing. In manage catagories admin can edit or delete the sub categories.



Products

With this setting admin can add new Products or can manage the existing products. Admin can edit, view, comment, review or delete the existing products. Along with this can also check the sales count, Net Revenue, mark the product as Active or Featured.



Edit Product : You can edit the product in two steps.

Edit Step 1 : In step 1 you can edit the name, product type, category, subcategory, URL name, Short description, tags, description, Product Details (Key, and Value), and SEO Details.


Edit Step 2 : In step 2 you can change product thumbnail image, preview and screenshots image, choose option to make product free for all, product pricing details, and File details (File Name, File External URL, File Upload, Price Assigment )


With Manage Product setting, admin can also view the products uploaded by author/vendor and can change its status by clicing on the Files option under Action




Users

This setting consist of the two sub - settings Add Users and User List.


Add Users

Admin can add user through this setting.


User List

Admin can manage the existing users through this setting.


Vendor Manager

This setting consist of the three sub - settings Consent Form Settings, List, and Vendor Request.


Consent Form Setting

Admin can customize the Consent Form, to Become an Author. Admin can add questions to show on consent form.


List

This setting allows admin to add vendor and manage the existing vendors.



Vendor Request

This setting shows vendor requests.

Email Section

This setting consist of the three sub - settings Integrations, Subscribed Email and Templates.


Integrations

This setting allows admin to connect the Autoresponders.


Subscribed Emails

Subscribed Email shows the list of emails that are subscribed.


Templates

In this setting admin can setup the SMTP settings, Basic Email setting, Registration Email, Forgot Password and Add New User Template.


Testimonial

With this setting admin can add testimonial and manage the existing testimonials.



Pages

With this setting, admin can customize the pages like About Us, Privacy policy, Terms and Conditions, and Authors Terms and Conditions as per need.


Contact Us

Admin can see the list of the user who contacted through contact form.



Revenue

Admin can select the currency, currency symbol, Commission Type, Commission, and Product Tax.


Transactions

Using this setting, admin can check the transaction details.



Coupon Settings

Coupon settings contains two sub -settings Add Coupon and Manage Coupons.

Add Coupns This setting enables you to add the coupon.


Manage Coupon This setting allows you to edit, or delete the coupons.



Language Management

Location setting consist of the rwo sub - settings Manage language and Add Language.


Manage language

With this, admin can add, edit or delete the languages as per need.


On clicking the view icon you will get a screen where you can change the text of the site as per your need. You can customize the text of frontend, admin message, frontend messagem and page title. After making all the changes click on the Update button so that all the changes gets reflected in one go.


Add Language

With this, admin can add new language.



Settings

Settings option includes basic option to customize your website like Payment Setting, Social Login Setting, Website Setting, Menu Setting, Advertisement, Media Setting, Home Content, Storage Settings, and Color Settings.


Payment Setting

Admin can choose the suitable payment method. We have provided Paypal, Stripe, RazorPay, Manual Transfer, Pawapay, and FlutterWave payment options


Social Login Settings

Through this setting, admin can add the details of Facebook, and Google for social login.


Website Settings

Admin can customize SEO settings, Site Images settings, Home Page settings and Footer Settings as per need.


With Site Image setting you can change site identity images like Site logo, favicon, preloader, etc.



Menu Settings

Admin can enable the sections that needs to be display on the frontend..


Advertisement Settings

Admin can customize the advertisement from this setting.


Media Settings

Admin can add the allowed file extensions, max file upload size, and max upload files (numeric value) for preview image.


Home Content

Admin can add and manage the home content with this setting. The content will show based on the type you select like Start Section or Why Choose Section.



Storage Settings

With the Storage Settings, admin can select the storage type as local, AWS, and Wasabi. To store data on AWS or Wasabi, it needs to add the respective key details.




Color Settings

With the Color Settings, admin can change Primary, Secondary, Text and Body color.


Wallet

Wallet settings allow admin to view User Wallet details, Withdraw Request, and Withdraw Setting.


User Wallet

Admin can see the user wallet history.


Withdraw Request

Admin can check the withdraw request of the authors. By clicking on edit icon, can change the status.



Withdrawl Settings

Admin can set the minimum withdrawl limit.


How to Become An Author

User can become Author by clicking on Become An Author option under Profile menu of User.


1) Click on the Become An Author option and fill consent form.


2) After clicking on Send Request button, you will get thank you page.


3) After the approval from Admin, author will be able to access the dashboard.


Author Dashboard

Author Dashboard comes with the various backend settings to flexibly appearance and operation of the website as well as can manage its customers, Products, sales and many more.


My Board

My Board settings contains the counts of Active Products, Free Products, Wallet Funds, Withdrawal Amount, Total Product Views and many more information. With this you can check the Product Views On Devices, Product Views From Browser,User Registration, and Order Sale in graphical format.



Products

With this setting, Author can add new Products or can manage the existing products. Admin can edit, view, comment, review the existing products. Along with this can also check the sales count, Net Revenue, mark the product as Active.



Customer

Author can see the details of the customer by clicking on View icon.


Sales

With this setting, author can see the sales/revnue details.


Wallet

Author can check the wallet history and can withdraw amount by clicking on the Withdraw button.


Account Settings

Admin can change the basic details and add bank account details.
